## Using the source

### You need install

- Nodejs    V12.18.3 or Higher
- Npm       V6.14.6 or Higher
- Chromium or Chrome

### Download && Execute
- Linux or Windows
> git clone https://github.com/Mastercliff/electron-music-player

> cd electron-music-player

> npm install

> npm start
