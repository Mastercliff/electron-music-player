## Using the app

### You need install

- Nodejs    V8.10.0 or Higher
- Npm       V3.5.2 or Higher
- Chromium  V78.0.3904.108 or Higher

### Download && Execute
- Linux
> git clone https://github.com/Mastercliff/electron-music-player

> cd electron-music-player

> electron .
